What is a burned down chart ?


A burn down chart is a graphical representation of work left to do versus time. The outstanding work is often on the vertical axis, with time along the horizontal. That is, it is a run chart of outstanding work.
It is useful for predicting when all of the work will be completed.

How is the data calculated on the burned down chart ?


Data on a burn down chart is a visual representation of estimated work time
and actual work time. The user inputs time estimates for a project so that
the program can compare actual work time with estimated time, while generating a recommended daily rate of work.

How do I design a workflow ?


To design a workflow, a user must input a task name.
For that specific task the user must input an estimated or desired amount of time to complete the objective within a certain amount of days.
This information will help the user keep an adequate work pace.

How is my data stored on the website ?


All data is stored locally on the website.
